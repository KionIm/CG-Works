# EVANGELION

Mayaによるモデリングとスキニング、Substance Painterでのテクスチャ制作、UE4でのリターゲットとサーバー接続を行いました。

現在は、プレイヤーがキャラクターを選択し、サーバーに接続して、適切にスポーン&同期させた段階です。
